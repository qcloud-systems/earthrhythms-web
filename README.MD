## ğŸŒ **Automatic Deployment (GitHub Pages + GitHub Actions)**

This project deploys automatically whenever you push to the **main** branch.

### 1. Enable GitHub Pages

In your repository:

**Settings â†’ Pages â†’ Build and deployment**

- **Source:** GitHub Actions

### 2. Add the GitHub Actions workflow

Create:

```
.github/workflows/deploy.yml
```

With:

```yaml
name: Deploy Vite site to GitHub Pages

on:
  push:
    branches: [ "main" ]
  workflow_dispatch:

permissions:
  contents: read
  pages: write
  id-token: write

jobs:
  build:
    runs-on: ubuntu-latest
    steps:
      - name: Checkout
        uses: actions/checkout@v4

      - name: Setup Node
        uses: actions/setup-node@v4
        with:
          node-version: 18

      - name: Install dependencies
        run: npm install

      - name: Build
        run: npm run build

      - name: Upload artifact
        uses: actions/upload-pages-artifact@v3
        with:
          path: ./dist

  deploy:
    needs: build
    runs-on: ubuntu-latest
    environment:
      name: github-pages
      url: ${{ steps.deployment.outputs.page_url }}

    steps:
      - name: Deploy to GitHub Pages
        id: deployment
        uses: actions/deploy-pages@v4
```

### 3. Deploy

Just push to `main`:

```bash
git add .
git commit -m "Update site"
git push
```

GitHub Actions will:

1. Install dependencies  
2. Build the Vite site  
3. Upload the `dist/` folder  
4. Deploy it to GitHub Pages  

Your live site will be available at:

```
https://qcloud-systems.github.io/earthrhythms-web/
```
